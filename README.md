📚 Мини-приложение «Подборка книг для любителей русского языка»

Мини-приложение с каталогом книг и корзиной для выбора и оформления заказа.
Проект предназначен для размещения в виде WebApp (SPA) и интеграции с платформой MAX.

Приложение позволяет:

просматривать подборку книг;

добавлять книги в корзину;

изменять количество товаров;

видеть итоговую сумму;

переходить к оформлению заказа.

🚀 Функциональность

📖 Каталог из 5 книг с изображениями, ценами и описанием

🛒 Корзина с подсчётом количества и общей суммы

💾 Сохранение корзины в LocalStorage

📱 Адаптация под мобильные устройства

⚡ Быстрая загрузка (SPA без серверной части)

🗂 Структура проекта
/index.html
/cart.css
/cart.js
/products.js
/README.md
